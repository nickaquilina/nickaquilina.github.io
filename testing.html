Testing & Debugging


Since the script is split into a lot of functions, I was able to test functions indipendently most of the time. Although I did not write Test Cases, I did use ad-hoc testing, with sample data for functions that supported such testing. Of course, my development was not Test Driven, in that I did not design the tests prior to programming.
